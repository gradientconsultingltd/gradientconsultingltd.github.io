# Gradient Consulting — Design Brainstorm

## Brief
One-page website for a UK-based tech recruitment and agentic AI consultancy. Dark, editorial, premium feel inspired by the Voyager reference (dark hero, bold typography, clean sections, strong contrast).

## Idea A: "Dark Editorial Brutalism"

**Design Movement:** Neo-Brutalism meets Editorial Dark Mode

**Core Principles:**
1. Raw typographic power — oversized display text dominates every section
2. Hard-edge geometry — no rounded corners, strict rectangular blocks
3. High-contrast monochrome base with a single electric accent (neon lime or electric blue)
4. Intentional asymmetry — text and imagery break the grid deliberately

**Color Philosophy:**
- Background: near-black `#0A0A0A`
- Surface: `#111111`, `#1A1A1A`
- Accent: Electric lime `#C8FF00` — signals innovation, disrupts the darkness
- Text: Pure white `#FFFFFF` and warm off-white `#E8E8E0`
- Emotional intent: Authority, disruption, forward-motion

**Layout Paradigm:**
- Sections split diagonally or with hard horizontal cuts
- Large left-aligned display headlines that overflow their containers
- Asymmetric two-column layouts where text and image weights are unequal

**Signature Elements:**
1. Oversized section numbers (01, 02, 03) in ultra-light weight as background texture
2. Horizontal rule lines that animate in on scroll
3. Monospace code snippets as decorative elements (referencing tech DNA)

**Interaction Philosophy:**
- Cursor becomes a crosshair on hover over CTAs
- Text links underline with a lime stroke that slides in from left
- Scroll-triggered section reveals with hard cut-in (no fade, just appear)

**Animation:**
- Hero headline: letters stagger in from bottom, 40ms delay each
- Section transitions: clip-path wipe from left edge
- Hover on cards: translate Y -4px, border turns lime

**Typography System:**
- Display: Space Grotesk Bold 700 — geometric, technical, authoritative
- Body: DM Mono Regular — monospace reinforces the tech/code identity
- Hierarchy: 96px / 48px / 24px / 16px / 12px

## Idea B: "Gradient Noir — Cinematic Dark Luxury"

**Design Movement:** Cinematic Dark Luxury / High-End SaaS

**Core Principles:**
1. Gradient as identity — the brand name "Gradient" is expressed through subtle colour transitions
2. Depth through layering — glassmorphism cards float above dark textured backgrounds
3. Restrained colour with purposeful warmth — dark navy/charcoal with amber-gold accents
4. Generous whitespace even in dark contexts — breathing room signals premium positioning

**Color Philosophy:**
- Background: Deep navy-black `#080C14`
- Surface: Semi-transparent glass panels `rgba(255,255,255,0.04)`
- Accent: Warm amber-gold `#F5A623` — signals value, expertise, warmth
- Gradient: Navy → deep purple `#0D1B2A → #1A0A2E` used in hero
- Emotional intent: Trust, intelligence, premium quality

**Layout Paradigm:**
- Full-width cinematic hero with gradient overlay on a dark image
- Floating glass cards for service offerings
- Staggered timeline layout for the process section
- Alternating text-left / image-right sections

**Signature Elements:**
1. Thin gold horizontal dividers between sections
2. Glass-effect cards with subtle border glow
3. Gradient text on key headlines (navy-to-purple-to-amber)

**Interaction Philosophy:**
- Glass cards tilt slightly on mouse move (3D perspective)
- CTA buttons have a shimmer sweep animation on hover
- Smooth scroll with momentum easing

**Animation:**
- Hero: Slow Ken Burns effect on background image
- Cards: Fade up with stagger on scroll entry
- Nav: Blur-in backdrop on scroll past hero

**Typography System:**
- Display: Playfair Display Bold — editorial gravitas, luxury feel
- Body: Inter Regular — clean, readable, professional
- Hierarchy: 80px / 44px / 22px / 16px / 13px

## Idea C: "Terminal Modernism — Dark Tech Precision" ← SELECTED

**Design Movement:** Dark Modernism with Terminal/Code DNA

**Core Principles:**
1. Precision over decoration — every element earns its place
2. The gradient is the brand — subtle colour gradients reference the company name and AI's multi-dimensional nature
3. Structural asymmetry — left-heavy layouts with deliberate negative space on the right
4. Tech credibility through typographic restraint — no decorative flourishes, only functional beauty

**Color Philosophy:**
- Background: Rich dark charcoal `#0E0E10` (not pure black — has depth)
- Surface: Slightly lighter `#16161A`, `#1C1C22`
- Primary Accent: Gradient from electric blue `#4F8EF7` to violet `#9B5CF6`
- Secondary: Warm white `#F0EEE8` for body text
- Muted: `#6B6B7A` for secondary text
- Emotional intent: Intelligence, precision, forward-thinking, trustworthy

**Layout Paradigm:**
- Off-centre hero: large headline left-aligned, visual element right — not centered
- Mosaic/bento-grid for services — unequal cell sizes create visual rhythm
- Horizontal scrolling stats bar between sections
- Full-bleed dark sections alternating with slightly lighter surface sections

**Signature Elements:**
1. Gradient underlines on key terms (blue-to-violet stroke)
2. Subtle dot-grid or circuit-board texture as section backgrounds (very low opacity)
3. Numbered section markers in gradient colour

**Interaction Philosophy:**
- Hover states reveal gradient borders on cards
- Navigation links have gradient underline that expands on hover
- CTAs pulse with a very subtle glow

**Animation:**
- Hero: Words slide up staggered, 50ms apart
- Stats: Count up animation when scrolled into view
- Cards: Scale from 0.97 to 1.0 with fade on scroll entry
- Gradient accent elements: slow hue-shift animation (very subtle)

**Typography System:**
- Display: Syne ExtraBold 800 — geometric, distinctive, modern without being trendy
- Subheadings: Syne Bold 700
- Body: Outfit Regular 400 — clean, slightly geometric, highly readable
- Mono accents: JetBrains Mono — for code/tech decorative elements
- Hierarchy: 88px / 52px / 32px / 18px / 14px

## Selected Approach: **Idea C — Terminal Modernism**

Rationale: The "Gradient" brand name is expressed literally through the blue-to-violet gradient motif. The dark charcoal base with precise typography signals both tech credibility and consultancy-level professionalism. The bento-grid services layout and asymmetric hero differentiate it from generic agency sites. Syne + Outfit is a distinctive pairing that avoids the overused Inter monoculture.
